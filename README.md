# Chat en tiempo real con Socket.io

Aplicación de chat en tiempo real desarrollada con Node.js, Express, Socket.io y Handlebars.

## Tecnologías
- Node.js
- Express
- Socket.io
- Handlebars

## Cómo correrlo
1. Clonar el repositorio
2. Instalar dependencias con `npm install`
3. Correr el servidor con `node src/index.js`
4. Abrir el navegador en `http://localhost:8080`
